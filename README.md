# Expense Tracker (Flask)

A full-stack web application that allows users to track daily expenses and visualize spending using charts.

## Features

* Add expenses
* Delete expenses
* Expense categories
* Total expense calculation
* Pie chart analytics

## Tech Stack

* Python
* Flask
* SQLite
* HTML / CSS
* Chart.js
